# Простой поисковик по тексту (TF-IDF Search Engine)

## Описание

**Мощный поисковый движок на Python** для индексации и поиска по текстовым файлам. 
Реализован алгоритм **TF-IDF** (Term Frequency - Inverse Document Frequency) для точного ранжирования релевантности результатов. 

Поддерживает:
- Обработку множества документов
- Токенизацию и нормализацию текста
- Удаление стоп-слов (NLTK)
- Стемминг для улучшения поиска
- Косинусное сходство для ранжирования

Идеально для локального поиска как **расширенный "Ctrl+F"** с учетом семантики![web:29][attached_file:1]

## Возможности

- Индексация текстовых файлов .txt в памяти
- Поиск по запросам с **ранжированием по TF-IDF**
- Токенизация, нижний регистр, удаление пунктуации
- **Стоп-слова** (русский/английский)
- **Стемминг** (SnowballStemmer)
- Вывод **топ-N релевантных документов** с оценками сходства
